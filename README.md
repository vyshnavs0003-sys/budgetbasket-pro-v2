# BudgetBasket

Live demo: [https://cosmic-croquembouche-be6178.netlify.app/]

A responsive grocery shopping web application that helps users manage their monthly grocery expenses while shopping.

## Features

- Browse products by category
- Search products in real time
- Add, update, and remove items from cart
- Set and track a monthly grocery budget
- Live remaining budget indicator
- Login and registration system
- Persistent session using localStorage
- Responsive design for mobile, tablet, and desktop

## Pages

- Home
- About
- Category
- Offers
- Contact
- Feedback
- Privacy Policy
- Terms and Conditions
- Cancellation and Refund
- My Orders *(routing only)*
- My Profile *(routing only)*

## Tech Stack

- React 18
- Vite
- Redux Toolkit
- Context API
- React Router
- Bootstrap Grid
- CSS3
- localStorage

## Folder Structure

- `assets/` – Images, icons, and other static files
- `components/` – Reusable UI components
- `context/` – Cart and budget logic
- `data/` – Product data
- `pages/` – Application pages
- `redux/` – Redux store and slices
- `styles/` – Global and page-specific styles
